Overview

Created by Bryce J. Meyer https://brycejmeyer.com - See License at bottom of page

This software is used to display a list of raffle prizes and winners. You may also display a slideshow and news ticker. This was designed in 2014 for use mainly with Google Chrome and Windows. See compatibility.

This software runs in 3 main parts. The Controls Page, Projector Page, and Server.

The Controls Page and Projector Page are opened in browser.

The Server runs in the background and enables communication between the Projector Page and the Controls Page. The server runs on NodeJS. See Server.

Basic Use

  • Double click Controls.html to open it in a browser
  • Double click start.cmd
  • At the top of the Controls page click the link to open "Projector Page"
  • Move Projector Page onto the projector's display
  • Press F11 on the Projector Page to enter full screen

Database

All settings and prizes are stored in the "database" file located in the root of this software. You can and should backup this file periodically when working with this software. If the file cannot be found a new one with defaults will be created. This file will only work with this program, however you can open it with any text editor. The file is JSON encoded UTF-8.

Compatibility

This software is portable and does not need to be installed. However all files should be extracted before using this software.

The Controls and Projector Pages are in html format and can be opened in a web browser. The pages were designed to work best in Google Chrome or Firefox. However other browsers might work.
All Operating Systems can work.

The Server runs on NodeJS. For windows the executable is included (/nodejs/node.exe). For Linux, the server requires you install NodeJS or Node. The server has been tested on Ubuntu using the "NodeJs" package.

For all systems "localhost" loopback must be available and no other programs running on port "6458".
